# Github

```c
git add .  

git commit -m "update" 

git push -f --set-upstream origin master:master

```

```c
git add .  

git commit -m "update" 

git push -f --set-upstream origin main:main

```

```c

git add .  && git commit -m "update" && git push -f --set-upstream origin main:main

```

使用git clean -df 可以清空本地缓存。

```c
git clean -df
```
